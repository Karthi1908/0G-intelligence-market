## 🌟 Vision
The **0G Intelligence Market** is a decentralized ecosystem for tokenizing, trading, and compositing "Intelligence" in the form of **iNFTs (Intelligent Non-Fungible Tokens)**. Built on the high-performance **0G Chain**, it leverages **0G Storage** for encrypted strategy metadata and **0G Compute** for verifiable execution.

## 🚀 Why This is the "Need of the Hour"
In the current AI-dominated landscape, intelligence is siloed. Proprietary trading algorithms, sophisticated machine learning models, and unique analytical skills are locked behind corporate walls or centralized platforms. 

**The problems we face:**
- **Lack of Ownership**: Creators cannot truly own and trade their algorithmic "soul."
- **Opaque Performance**: "Black box" strategies lack verifiable history.
- **Single-Point Failure**: Centralized marketplaces can censor or front-run successful strategies.
- **Stagnant Intelligence**: There is no easy way to combine different "brains" into a superior entity.

**0G Intelligence Market solves this by providing a trustless, verifiable, and composable layer for intelligence.**

## 🧬 How It Works (The Core Architecture)

### 1. Tokenized Intelligence (ERC-7857)
Every strategy is minted as an **iNFT** compliant with the **ERC-7857** standard. Unlike static NFTs, iNFTs represent **dynamic, executable logic**. 
- **Encrypted Metadata**: The strategy's internal weights, logic, and sensitive data are stored encrypted on **0G Storage**.
- **On-Chain Identity**: The NFT acts as a secure key for authorization and ownership.

### 2. Strategy Composability (Merging)
The breakthrough feature of the platform is **Intelligence Merging**. 
- Users can combine two different iNFTs (e.g., an ARIMA model and an LSTM neural net) to create a **Composite iNFT**.
- The new entity runs both strategies in parallel, aggregating signals to provide higher conviction and reduced drawdown.

### 3. Verifiable Execution
Using a decentralized **Oracle** system (simulated in Galileo Testnet), every prediction or trade signal generated by an iNFT can be verified. This ensures that the "intelligence" you buy is actually performing as advertised.

## 📦 Getting Started

### Prerequisites
- Node.js & npm
- MetaMask (connected to 0G Galileo Testnet)
- 0G Testnet Tokens

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Karthi1908/0G-intelligence-market.git
   cd 0G-intelligence-market
   ```
2. Install dependencies:
   ```bash
   npm install
   cd frontend && npm install
   ```
3. Set up environment variables in `.env` (see `.env.example`).

### Deployment
1. Deploy smart contracts:
   ```bash
   npx hardhat run --network og-testnet contracts/scripts/deploy.js
   ```
2. Start the frontend:
   ```bash
   cd frontend
   npm run dev
   ```
